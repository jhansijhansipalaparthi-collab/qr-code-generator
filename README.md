# QR Code Generator 
Python tool to generate QR codes from user details. Built for  quickly capture customer information.

# Features
- Takes Name, Age, Email, Mobile as input from user
- Generates QR code instantly using `qrcode` library
- Saves output as `mydetails_in_qr.png` 

# How to Run This Project
1. Install required library: `pip install qrcode`
2. Run the script: `qrgeneator.py`
3. Enter customer details when prompted
4. Scan `mydetails_in_qr.png` to get all details

# Tech Stack
- Python 3
- qrcode library

# QR code Use Case
Instead of typing long details manually,users can share info via QR code quickly
